# countdown

使用 canvas 实现一款酷炫的倒计时web应用  

## 介绍  

主要功能 : 倒计时  
特点 :  

1. 用小球绘制数字  
2. 当时间变化，数字对应的小球进行散落  
3. 小球弹跳  

看起来很酷炫 ...
